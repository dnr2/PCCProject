##Processamento de Cadeias de Caracteres

Este repositório contém códigos dos projetos requeridos para a disciplina de Processamento de cadeias de caracteres 2014.2

###TPF

The Pattern Finder (TPF) é uma ferramenta de busca de padrões que implementa conhecidos algoritmos da área, Incluindo:

Boyer-Moore

Aho-Corasick

Wu-Mamber

Sellers

###IPMT

Indexed Pattern Matching Tool (IPMT) é uma ferramenta de criação de índices com compressão e busca de padrões que implementa conhecidos algoritmos da área, Incluindo:

Árvore de sufixos

Array de sufixos

LZW (algoritmo de compressão)
